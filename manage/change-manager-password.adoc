= Change ONTAP tools Manager password
:icons: font
:imagesdir: ../media/

[.lead]
You can change the administrator password using ONTAP tools Manager.

.Steps

. Launch ONTAP tools Manager from a web browser: `\https://<ONTAPtoolsIP>:8443/virtualization/ui/` 
. Log in with the ONTAP tools for VMware vSphere administrator credentials you provided during deployment.
. Select the *administrator* icon on the top right corner of the screen and select *Change password*.
. In the change password pop-up window, enter the old password and the new password details. The constraint for changing the password is displayed on the user interface screen.
. Select *Change* to implement the changes.
